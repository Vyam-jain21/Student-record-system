# Student Record System (C Project)

A simple Student Record Management System written in C using file handling. This program allows users to add, delete, search, and display student records using a menu-driven interface.

## Features
- Add new student
- Delete student by roll number
- Search student
- Display all students
- Uses structures and binary file handling

## Technologies Used
- C Language
- File I/O (fread, fwrite, fopen, remove, rename)
- Menu-driven programming

## How to Run
1. Compile the code using a C compiler:
   `gcc filename.c -o output`
2. Run the executable:
   `./output`
